# Fokus v2 - Temporizador Pomodoro com To-Do List

**Fokus** é um temporizador Pomodoro projetado para aumentar a produtividade durante os estudos ou atividades de trabalho, com o diferencial de uma **To-Do List** integrada para organizar e gerenciar suas tarefas. Agora, você pode acompanhar suas atividades enquanto se concentra em períodos de foco e realiza intervalos de descanso, tudo em um só lugar!

## Funcionalidades
- **Temporizador Pomodoro**
  - **25 minutos de foco**: Período de trabalho sem interrupções.
  - **5 minutos de descanso curto**: Intervalo rápido para recarregar as energias.
  - **15 minutos de descanso longo**: Intervalo mais longo após 4 ciclos de foco.
- **Contagem Regressiva Visual:** Interface simples com visualização clara dos tempos de foco e intervalo, além das tarefas listadas.
- **Música de Fundo:** Ative uma música suave de fundo para ajudar na concentração enquanto você trabalha no seu temporizador Pomodoro.
- **Alertas Sonoros:** Sons ao término de cada período (foco ou intervalo), para avisar o usuário quando é hora de passar para a próxima etapa.

- **To-Do List Integrada**
  - **Adicionar Tarefas**: Crie tarefas facilmente para organizar seu trabalho.
  - **Marcar como Concluída**: Marque tarefas como feitas quando concluir um item.
  - **Deletar Tarefas**: Remova tarefas da lista à medida que forem completadas ou descartadas.
  - **Organização Eficiente**: Tenha suas tarefas sempre visíveis ao lado do temporizador para uma gestão mais eficaz.

## Como Usar

1. **Iniciar o Temporizador**: Clique no botão para iniciar o ciclo Pomodoro (25 minutos de foco).
2. **Gerenciar Tarefas**:
   - **Adicionar Tarefa**: Insira uma nova tarefa na lista clicando no botão "Adicionar Tarefa".
   - **Marcar como Concluída**: Clique ao lado de uma tarefa para marcá-la como concluída.
   - **Deletar Tarefa**: Exclua tarefas clicando no ícone de deletar ao lado de cada item.
3. **Descanse com Intervalos**: Após os 25 minutos de foco, o temporizador começará um intervalo curto de 5 minutos. Após quatro ciclos, será iniciado um intervalo longo de 15 minutos.
4. **Música de Fundo**: Ative a música para ajudar a manter o foco. Você pode ligar ou desligar a qualquer momento.
5. **Alertas Sonoros**: Quando o tempo de foco ou intervalo terminar, um alerta sonoro será disparado para avisar sobre a próxima etapa.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da interface.
- **CSS3**: Estilos para layout e design visual.
- **JavaScript**: Lógica do temporizador, To-Do List, contagem regressiva e interatividade.
- **Web Audio API**: Implementação de música de fundo e alertas sonoros.
- **LocalStorage**: Armazenamento de tarefas para persistência entre as sessões.
